Learning by example (15 April 2021)

Trying to (roughly) recreate the example using HTML tags and CSS styling. This blogpost has some content on the left, but as you can see, there’s also a contacts sidebar on the right. You could try make those links clickable.

Ideally, if I scroll down in this post, the contacts bar should just stay put.

To recreate this page, you’ll notice that it’s divided into several parts (e.g. a big header on top, this blog post on the left and the contact info on the right). The starting example has div tags to divide the content up. You’ll need to work out how to apply styling to each div (try using a class attribute).

Often, if you resize the window, things will suddenly look different. Try make your page look nice no matter the window size.
